# OHP Management App

This is a web application designed to manage Occupational Health Program (OHP) data, including campus buildings, laboratories, principal investigators (PIs), lab members, and OHP forms. It is built with Flask for the backend, PostgreSQL for the database, and HTML/CSS for the frontend.

## Features

- Manage campus buildings and laboratories.
- Record principal investigators and lab members.
- Store and retrieve OHP form data.
- User-friendly interface for Environmental Health and Safety (EHS) personnel.

## Tech Stack

- **Frontend**: HTML, CSS
- **Backend**: Python (Flask)
- **Database**: PostgreSQL

## Requirements

- Python 3.12.7
- PostgreSQL
